# Career Website

## Overview
A Flask-based careers website. Currently displays a simple "Hello, World" message as a starting point.

## Project Structure
- `app.py` - Main Flask application entry point

## Tech Stack
- Python 3.11
- Flask web framework
- Gunicorn (production server)

## Development
Run the app with:
```bash
python app.py
```
The development server runs on port 5000.

## Deployment
Production deployment uses Gunicorn and is configured for autoscaling.
